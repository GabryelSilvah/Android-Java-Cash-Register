# Cash Register

## Sobre
Inicialmente, esse é um projeto desenvolvido para uma disciplina enquanto eu ainda estava na faculdade. Relativamente simples, consiste 
basicamente em registrar uma lista de produtos a serem comprados. Ele tem algumas outras funcionalidades, por exemplo, 
uma lista detalhada dos preços de cada produto e montante final e possue tradução para três idiomas (Português, Espanhol e Inglês).
Mas, como eu bem disse, é um projeto básico, no entanto, demonstra conhecimentos razoáveis sobre desenvolvimento mobile para Android.

## Funcionalidades
- Listagem, alteração e exclusão de produtos
- Detalhes do produto adicionado
- Filtros de listagem
- Tabela detalhada de valores
- Tradução para 3 idiomas
